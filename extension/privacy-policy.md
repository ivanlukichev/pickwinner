# PickWinner Tools Extension Privacy Policy

Effective date: March 26, 2026

This privacy policy applies to the PickWinner Tools browser extension builds in this folder, including the Chrome, Opera, and Firefox versions.

## What the extension does

The current extension is a lightweight Coin Flip tool.

It allows the user to:

- open a popup from the browser toolbar
- flip a virtual coin
- receive a Heads or Tails result
- open PickWinner Tools website links from the popup

## Data the extension does not collect

The extension is designed to run locally and does not intentionally collect, store, or transmit personal information such as:

- name
- email address
- phone number
- payment information
- browsing history
- page contents from other websites
- keystrokes outside the extension interface
- contacts
- location data

## Local storage inside the extension

The extension may store a small amount of local data inside the browser to improve the popup experience.

At the moment, this is limited to the last displayed coin flip result so the popup can keep its previous state.

This data stays inside the browser extension environment and is not intended to be sent to PickWinner Tools servers by the extension itself.

## Network activity

The extension popup is designed to avoid background tracking and does not include analytics scripts inside the extension package.

Network activity may happen only when you intentionally open a PickWinner Tools page from the popup, such as:

- the PickWinner Tools homepage
- future tool pages linked by the extension

When you open the website from the extension, the website is governed by the site privacy policy and site behavior of pickwinner.tools.

## Permissions

The extension is built to use the minimum behavior needed for a toolbar popup and its own packaged files.

It is not designed to:

- inspect unrelated websites
- read browsing history
- monitor page activity
- inject scripts into websites

## Children

PickWinner Tools may be used by general audiences, including classrooms and families, but the extension is not designed to collect personal information from children.

## Changes to this policy

This policy may be updated if the extension later adds storage, sync, analytics, accounts, permissions, or additional browser features.
