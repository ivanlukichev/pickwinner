# Chrome Web Store Notes

## Listing name

PickWinner Tools Coin Flip

## Short summary

Flip a clean virtual coin directly from the browser toolbar.

## Description

PickWinner Tools Coin Flip is a lightweight browser extension for quick Heads or Tails decisions.

Features:

- one-click popup access
- clean coin flip animation
- simple Heads or Tails result
- lightweight interface
- link to PickWinner Tools for more random tools

## Permissions

No extra browsing permissions requested.

## Privacy

- Privacy policy: [`../privacy-policy.md`](../privacy-policy.md)
- No account required
- No extension-side analytics
- No reading of page content or browsing history
