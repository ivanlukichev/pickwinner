# Firefox Add-ons Notes

## Listing name

PickWinner Tools Coin Flip

## Summary

Minimal coin flip extension with a polished popup and no unnecessary permissions.

## Main points

- Firefox package lives in [`../firefox`](../firefox)
- includes `browser_specific_settings.gecko`
- popup works as a local extension UI
- homepage link sends users to PickWinner Tools
