# Extension Submission Checklist

Before publishing the PickWinner Tools extension packages:

- verify all manifests use the correct name, version, and icons
- confirm popup opens and flips correctly in Chrome
- confirm popup opens and flips correctly in Edge
- confirm popup opens and flips correctly in Opera
- confirm popup opens and flips correctly in Firefox
- confirm the footer link opens `https://pickwinner.tools/`
- confirm no unnecessary permissions are present
- confirm the privacy policy matches current behavior
- prepare screenshots of the popup for store listings
- prepare short and long descriptions for each store
- package each browser folder separately for upload
